# 🦀 Rust CLI Todo App – Feature Checklist

A complete roadmap for building a powerful CLI-based todo application in Rust to learn rust.

---

## ✅ Core Features (MVP)

* [ ] Add task
* [ ] Edit task
* [ ] Delete task
* [ ] Mark task as complete / incomplete
* [ ] List all tasks
* [ ] Assign unique task IDs
* [ ] Persistent storage (JSON / file-based)
* [ ] Basic CLI commands (`add`, `list`, `done`, `delete`)
* [ ] Help menu (`--help`)

---

## ⚙️ Intermediate Features

* [ ] Due dates & times
* [ ] Task priorities (low, medium, high)
* [ ] Tags / categories
* [ ] Search tasks
* [ ] Filter tasks (by tag, status, date)
* [ ] Sort tasks (date, priority)
* [ ] Config file support
* [ ] Colored terminal output

---

## 🧠 Advanced Features

* [ ] Recurring tasks (daily, weekly, etc.)
* [ ] Subtasks / nested tasks
* [ ] Undo / redo functionality
* [ ] Task dependencies
* [ ] Task descriptions / notes
* [ ] Attach links or files
* [ ] Bulk operations

---

## 💾 Storage & Sync

* [ ] Support multiple backends:

  * [ ] JSON
  * [ ] SQLite

* [ ] Auto-save system
* [ ] Backup system
* [ ] Optional sync:

  * [ ] Git-based sync
  * [ ] Cloud sync
